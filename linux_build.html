Creating minimal operating system

Building linux from scratch: Git clone the repository https://github.com/arjuncr/os-from-scratch.git and run ./build_cross_gcc.sh then run ./build_image.sh,this will create image tree and the project is underdevelopment... Basic os building: Git clone the repository https://github.com/arjuncr/base_os_build_script.git and run ./build_linux.sh, this will generate bootable iso, it is minimal os. Building light-os: Git clone https://github.com/arjuncr/light-os-buildtools.git and run download.sh (for getting source) and the run build_linux.sh, this will generate iso image. Custom debian: Git clone https://github.com/arjuncr/custom_debian.git, run custom_debian.sh, this will generate live debian image. Docker : Git clone https://github.com/arjuncr/docker.git the run ./install_docker.sh for installing docker in developmemt sysem, run ./build_docker.sh for building the docker and run ./docker_run.sh -> running the docker.
